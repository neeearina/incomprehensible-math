# Идея задачи: переход от матрицы смежности к матрице инцидентности

Граф задан **матрицей смежности** — квадратной таблицей, где на пересечении строки `i` и столбца `j` стоит:

- `1`, если между вершинами `i` и `j` существует ребро,
- `0`, если ребра нет.

По такой матрице можно закодировать как неориентированные, так и ориентированные рёбра:

- если `A[i][j] = 1` и `A[j][i] = 1` (при `i ≠ j`), будем считать, что между вершинами `i` и `j` есть **неориентированное** ребро;
- если `A[i][j] = 1` и `A[j][i] = 0`, будем считать, что есть **ориентированное** ребро `i → j`.

Задача программы — по данной матрице смежности построить **матрицу инцидентности**.

Матрица инцидентности — это таблица размера `n × m`, где:

- `n` — число вершин,
- `m` — число рёбер.

Строки соответствуют вершинам, столбцы — рёбрам.  
Для каждого ребра:

- если ребро **неориентированное** между вершинами `u` и `v`, в соответствующем столбце матрицы инцидентности ставим:
  - `1` в строке вершины `u`,
  - `1` в строке вершины `v`;

- если ребро **ориентированное** `u → v`, то:
  - `-1` в строке вершины-начала `u`,
  - `+1` в строке вершины-конца `v`.
