# Идея задачи: определение, является ли граф деревом, и вычисление его высоты

Граф задан **матрицей весов** — квадратной таблицей `n × n`, где:

- `0` на диагонали означает отсутствие петли,
- `0` вне диагонали означает отсутствие ребра,
- положительное число означает наличие ребра (его вес).

Вес ребра для задачи неважен — важно лишь, есть ребро или нет.  
Граф считается **неориентированным**, поэтому матрица должна быть симметричной.

## Когда граф является деревом?

Неориентированный граф является **деревом**, если одновременно выполняются два условия:

1. **Граф связный**, то есть из любой вершины можно добраться до любой другой.
2. **В графе нет циклов**.  
   В неориентированном графе это равносильно тому, что количество рёбер `m = n − 1`.

Если хотя бы одно условие нарушено, граф **не является деревом**.

## Что такое высота дерева?

Высота дерева — это максимальное расстояние в рёбрах от выбранного корня до любой вершины.  
В данной задаче корень фиксирован: **вершина 1** (индекс 0 в программе).

Высота вычисляется с помощью обхода в ширину (BFS), который находит расстояние от корня до всех других вершин. Максимальное из этих расстояний и есть высота дерева.
