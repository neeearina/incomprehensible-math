# Идея задачи: радиус и диаметр графа

Рассмотрим неориентированный невзвешенный граф, где каждое ребро имеет одинаковую стоимость прохода.

**Кратчайшее расстояние** между вершинами `u` и `v` — минимальное число рёбер в пути между ними.

**Эксцентриситет** вершины `v` — максимальное расстояние от `v` до любой другой вершины:

```
e(v) = max_u d(v, u)
```

**Диаметр** графа — максимальный эксцентриситет среди всех вершин:

```
D = max_v e(v)
```

**Радиус** графа — минимальный эксцентриситет:

```
R = min_v e(v)
```

Чтобы вычислить радиус и диаметр, нужно знать все кратчайшие расстояния между парами вершин. Для невзвешенного графа это удобно делать с помощью обхода в ширину (BFS), запуская его из каждой вершины.
