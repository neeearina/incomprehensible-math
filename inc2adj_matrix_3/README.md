# Идея задачи: переход от матрицы инцидентности к матрице смежности

Граф задан **матрицей инцидентности** — таблицей размера `n × m`, где:

- `n` — число вершин,
- `m` — число рёбер.

Строки соответствуют вершинам, столбцы — рёбрам.  
Каждый элемент матрицы может принимать значения `-1`, `0` или `1`.

Интерпретация столбцов (рёбер):

- если в столбце есть **две вершины с значением `+1`** и нет `-1`, то это
  **неориентированное** ребро между этими двумя вершинами;
- если в столбце есть **одна вершина с `-1` и одна с `+1`**, то это
  **ориентированное** ребро: от вершины с `-1` к вершине с `+1`.

Наша задача — по такой матрице построить **матрицу смежности**.

Матрица смежности — это квадратная матрица `n × n`, где на пересечении строки `i`
и столбца `j` стоит:

- `1`, если из вершины `i` в вершину `j` есть ребро,
- `0`, если ребра нет.

Правила заполнения матрицы смежности:

- для **неориентированного** ребра между `u` и `v` в матрице смежности
  ставим `1` в ячейках `(u, v)` и `(v, u)`;
- для **ориентированного** ребра `u → v` ставим `1` только в ячейке `(u, v)`.

Алгоритм работы программы:

1. Программа считывает матрицу инцидентности из текстового файла.
2. Проверяет корректность данных:
   - все строки одной длины;
   - элементы только из множества `{-1, 0, 1}`;
   - каждый столбец соответствует либо ориентированному, либо неориентированному ребру.
3. Для каждого столбца определяет тип ребра (ориентированное или неориентированное)
   и вершины, которые это ребро соединяет.
4. По полученному списку рёбер заполняет матрицу смежности.
5. На экран выводятся:
   - исходная матрица инцидентности;
   - список рёбер с указанием их типа;
   - полученная матрица смежности в удобочитаемом виде.
