# Идея задачи: нерекурсивные алгоритмы BFS и DFS

Граф задаётся **матрицей весов**, где:
- `0` — ребра нет,
- положительное число — ребро существует (его вес не важен для поиска).

Из матрицы весов строится **список смежности**, который удобен для обходов:
каждой вершине соответствует список соседей, в которые из неё есть ребро.

## Алгоритм поиска в ширину (BFS)

Поиск в ширину исследует граф слоями.  
В нерекурсивной реализации используется очередь:

1. Поместить стартовую вершину в очередь.
2. Пока очередь не пуста:
   - извлечь вершину;
   - отметить посещённой;
   - добавить всех непосещённых соседей в очередь.
3. В порядке извлечения вершин формируется очередность обхода.

## Алгоритм поиска в глубину (DFS)

Поиск в глубину уходит как можно дальше вдоль одного пути.  
В нерекурсивной форме используется стек:

1. Поместить стартовую вершину в стек.
2. Пока стек не пуст:
   - извлечь вершину;
   - если она не посещена — отметить её;
   - добавить непосещённых соседей в стек.
3. Порядок извлечения задаёт последовательность обхода.

## Итог работы программы

Программа:
1. считывает матрицу весов из файла;
2. проверяет корректность данных;
3. формирует список смежности;
4. выполняет нерекурсивные BFS и DFS из выбранной вершины;
5. выводит порядок обхода вершин.
