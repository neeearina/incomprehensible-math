# Идея задачи: проверка графов на изоморфизм по матрицам смежности

Граф задаётся **матрицей смежности** — квадратной таблицей, в которой элемент `A[i][j]` равен `1`, если между вершинами `i` и `j` существует ребро, и `0` — если ребра нет. Два графа считаются **изоморфными**, если их структуры совпадают с точностью до переименования вершин. Это означает, что существует некоторая перестановка вершин одного графа, превращающая его матрицу смежности в матрицу смежности другого.

Формально графы изоморфны, если существует биекция `p` такая, что для всех `i` и `j` выполняется:

`A[i][j] = B[p[i]][p[j]]`
