# Идея задачи: матрица достижимости

Граф задан **матрицей смежности** — квадратной таблицей `n × n`, где элемент `A[i][j]` равен:

- `1`, если из вершины `i` в вершину `j` есть ребро,
- `0`, если ребра нет.

Граф может быть ориентированным, поэтому матрица смежности не обязана быть симметричной.

**Матрица достижимости** показывает, можно ли добраться из одной вершины в другую по пути из одного или нескольких рёбер.  
Элемент матрицы достижимости `R[i][j]` равен:

- `1`, если из вершины `i` существует путь в вершину `j`,
- `0`, если пути нет.

При этом вершина всегда считается достижимой сама из себя (путь длины 0), поэтому на диагонали матрицы достижимости стоят единицы.

Чтобы построить матрицу достижимости:

1. Программа считывает матрицу смежности из текстового файла и проверяет корректность данных
   (матрица квадратная, содержит только `0` и `1`).
2. Для каждой вершины `v` запускается обход в ширину (BFS), который находит все вершины,
   достижимые из `v`.
3. Для каждой пары вершин `(i, j)` заполняется матрица достижимости:
   если `j` была посещена при BFS из `i`, то `R[i][j] = 1`, иначе `R[i][j] = 0`.
4. На экран выводятся исходная матрица смежности и построенная матрица достижимости
   в удобочитаемом виде.
